openMairie Exemple - Gestion des thèmes jquery-ui
=================================================

openMairie Exemple utilise plusieurs thèmes jquery-ui spécifiques, lors de la
mise à jour de cette librairie, il faut effectuer une procédure particulière
dont voici les principales instructions

* Aller sur le site de jquery-ui : "http://jqueryui.com/themeroller"

Thème "overcast"
----------------

* Sélectionner le thème "overcast"

* Changer les valeurs dans "Font Settings"

- Size : 11px

* Puis cliquer sur "Download" pour télécharger l'archive

* Extraire l'archive, et renommer le dossier "custom-theme" par "om_overcast"

* Copier le contenu du dossier "om_overcast" dans openmairie_exemple
  "lib/jquery-ui/css/om_overcast"

Thème "sunny"
-------------

* Sélectionner le thème "sunny"

* Changer les valeurs dans "Font Settings"

- Size : 11px

* Puis cliquer sur "Download" pour télécharger l'archive

* Extraire l'archive, et renommer le dossier "custom-theme" par "om_sunny"

* Copier le contenu du dossier "om_sunny" dans openmairie_exemple
  "lib/jquery-ui/css/om_sunny"

Thème "ui-darkness"
-------------------

* Sélectionner le thème "ui-darkness"

* Changer les valeurs dans "Font Settings"

- Size : 11px

* Changer les valeurs dans "Content"

- Background-color: #ffffff
- Text : #434343

* Puis cliquer sur "Download" pour télécharger l'archive

* Extraire l'archive, et renommer le dossier "custom-theme" par "om_ui-darkness"

* Copier le contenu du dossier "om_ui-darkness" dans openmairie_exemple
  "lib/jquery-ui/css/om_ui-darkness"

Manipulation commune
--------------------

* Copier le contenu du dossier "js" dans openmairie_exemple "lib/jquery-ui/js"

* Vérifier dans "obj/utils.class.php" la version de jquery et jquery-ui

* Supprimer les anciens fichiers désormais inutiles
